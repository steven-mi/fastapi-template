# FastAPI Template

This template enables you to easily bootstrap a FastAPI project. The template contains:
- FastAPI
- Poetry
- GitHub Action for CICD
- Devcontainer
- Linter and others

## Getting Started

### Start FastAPI Application
```
uvicorn src.main:app  --reload 
```
